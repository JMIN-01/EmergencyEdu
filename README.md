## 왜 만들었나요 ?
신고의무자가 소속된 기관·시설 등의 장(이하 ‘기관·시설장’)은 소속 신고의무자에게 매년 1시간 이상 신고의무자 교육을 실시하고, 그 결과를 관계 행정기관의 장에게 제출해야 합니다.
<br>신고의무자교육 결과보고를 제출할 때는 이메일이나 팩스를 사용하여 제출합니다.
