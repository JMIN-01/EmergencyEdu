## 왜 만들었나요 ?
신고의무자가 소속된 기관·시설 등의 장(이하 ‘기관·시설장’)은 소속 신고의무자에게 매년 1시간 이상 신고의무자 교육을 실시하고, 그 결과를 관계 행정기관의 장에게 제출해야 합니다.
<br>신고의무자교육 결과보고를 제출할 때는 서식을 출력 후 작성, 스캔하여 시청(보건소)에 제출 형식으로 진행됩니다.
<br>이제 우리는 종이 출력을 최소화하여, 웹 하나로 제출하고자 합니다.<br><br>

<strong>이 웹은 다음과 같이 사용합니다.</strong>

- <strong>의료기관</strong>은 기관정보 및 교육 결과를 입력 후 제출합니다.
![의료기관](assets/member.gif)<br><br>
- <strong>시청(보건소)</strong>은 실시간으로 교육 결과 제출 현황을 확인하고, 상태값을 변경할 수 있습니다.
![기관장](assets/admin.gif)<br>
